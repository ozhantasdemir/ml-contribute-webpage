Code

ML Tools is a collection of three complementary parts: ML Interface Wizard, ML Measure, and ML Contribute. These tools are designed to simplify the development and deployment of Machine Learning models.

Abstract

This project aims to address the challenge of making machine learning (ML) more accessible to a wider audience and improving the process of labeling unlabeled data. The objective is to develop a solution that automates the training of any tabular dataset and confidently determines the labels of unlabeled data. The project begins with an introduction to the field of ML and highlights the problem statement, which revolves around the need to democratize ML and streamline the labeling process for unlabeled data. Traditional ML approaches often require significant technical expertise and manual effort, limiting their accessibility to a broader range of users. To tackle this problem, a ML model has been created, capable of automatically preprocessing any dataset through feature selection, scaling, and encoding techniques. This model simplifies the data preparation phase, reducing the burden on users and making ML more approachable. Additionally, a voting-based algorithm has been developed to determine the labels of unlabeled data. Leveraging the expertise of domain professionals, this algorithm combines multiple predictions to achieve a confident and reliable labeling process. By providing accurate labels for unlabeled data, the solution enhances the overall reliability of predictions made by ML models. The results of this project demonstrate its versatility, as the proposed algorithm can be applied to any tabular dataset. The effectiveness of the solution is dependent on the quality of the dataset and the information obtained from domain experts. In experimental evaluations, the solution achieved an improvement of up to 10% in accuracy.
